Problem 1: Palindrome Checker (Python)

Easy

Write a Python function that checks if a given string is a palindrome (reads the same backward as forward). The function should return True if it's a palindrome and False otherwise.

def is_palindrome(s):
    """Check if a string is a palindrome."""
    # Remove non-alphanumeric characters and convert to lowercase
    cleaned = ''.join(c for c in s if c.isalnum()).lower()
    return cleaned == cleaned[::-1]

# Examples:
print(is_palindrome("A man, a plan, a canal: Panama"))  # True
print(is_palindrome("racecar"))  # True
print(is_palindrome("hello"))  # False

Explanation:

This solution first cleans the input string by removing all non-alphanumeric characters and converting it to lowercase. Then it checks if the cleaned string is equal to its reverse (using Python's slice notation [::-1] to reverse the string).

Problem 2: Two Sum (Java)

Easy

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target. You may assume that each input would have exactly one solution, and you may not use the same element twice.

import java.util.HashMap;
import java.util.Map;

public class TwoSum {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> numMap = new HashMap<>();
        for (int i = 0; i < nums.length; i++) {
            int complement = target - nums[i];
            if (numMap.containsKey(complement)) {
                return new int[] { numMap.get(complement), i };
            }
            numMap.put(nums[i], i);
        }
        throw new IllegalArgumentException("No two sum solution");
    }
}

Explanation:

This Java solution uses a HashMap to store the complement of each number (target - current number) as we iterate through the array. When we find a number whose complement is already in the map, we return the indices. This approach has O(n) time complexity.

Problem 3: Interactive To-Do List (JavaScript)

Medium

Create a simple interactive to-do list application with vanilla JavaScript that allows users to:

  • Add new tasks with a text input
  • Mark tasks as complete
  • Delete tasks
  • Persist tasks between page reloads using localStorage

// HTML Structure
<div id="todo-app">
    <h2>To-Do List</h2>
    <input type="text" id="task-input" placeholder="Add a new task..." />
    <button id="add-btn">Add Task</button>
    <ul id="task-list"></ul>
</div>

// JavaScript Solution
document.addEventListener('DOMContentLoaded', () => {
    const taskInput = document.getElementById('task-input');
    const addBtn = document.getElementById('add-btn');
    const taskList = document.getElementById('task-list');
    
    // Load tasks from localStorage
    loadTasks();
    
    addBtn.addEventListener('click', addTask);
    taskInput.addEventListener('keypress', (e) => {
        if (e.key === 'Enter') addTask();
    });
    
    function addTask() {
        const taskText = taskInput.value.trim();
        if (taskText) {
            createTaskElement(taskText);
            saveTask(taskText);
            taskInput.value = '';
        }
    }
    
    function createTaskElement(taskText, isCompleted = false) {
        const li = document.createElement('li');
        li.innerHTML = `
            <input type="checkbox" ${isCompleted ? 'checked' : ''} />
            <span class="${isCompleted ? 'completed' : ''}">${taskText}</span>
            <button class="delete-btn">×</button>
        `;
        
        const checkbox = li.querySelector('input[type="checkbox"]');
        const deleteBtn = li.querySelector('.delete-btn');
        
        checkbox.addEventListener('change', () => {
            li.querySelector('span').classList.toggle('completed');
            updateStorage();
        });
        
        deleteBtn.addEventListener('click', () => {
            li.remove();
            updateStorage();
        });
        
        taskList.appendChild(li);
    }
    
    function saveTask(taskText) {
        const tasks = JSON.parse(localStorage.getItem('tasks') || '[]');
        tasks.push({ text: taskText, completed: false });
        localStorage.setItem('tasks', JSON.stringify(tasks));
    }
    
    function loadTasks() {
        const tasks = JSON.parse(localStorage.getItem('tasks') || '[]');
        tasks.forEach(task => {
            createTaskElement(task.text, task.completed);
        });
    }
    
    function updateStorage() {
        const tasks = [];
        document.querySelectorAll('#task-list li').forEach(li => {
            tasks.push({
                text: li.querySelector('span').textContent,
                completed: li.querySelector('input').checked
            });
        });
        localStorage.setItem('tasks', JSON.stringify(tasks));
    }
});

Explanation:

This JavaScript solution creates a complete to-do list application with persistence. It demonstrates DOM manipulation, event handling, and localStorage usage. The solution includes adding tasks, marking them as complete, deleting tasks, and saving the state between page reloads.

Python Documentation

Variables and Data Types

Python has several built-in data types:

  • int - integer numbers
  • float - floating point numbers
  • str - strings of characters
  • bool - True or False values
  • list - ordered, mutable collections
  • tuple - ordered, immutable collections
  • dict - key-value pairs
  • set - unordered collections of unique elements

# Variable assignment examples
age = 25  # int
price = 19.99  # float

user = {"name": "John", "age": 30}  # dict
unique_numbers = {1, 2, 3}  # set

Functions and Modules

Functions are defined using the def keyword. Python supports modular programming through modules and packages.

# Function definition
def greet(name):
    """This function greets the person passed in as parameter"""
    return f"Hello, {name}!"

 # Using a function
print(greet("World"))  # Output: Hello, World!

# Importing modules
import math
print(math.sqrt(16))  # 4.0

from datetime import date
today = date.today()
print(today)  # Current date

Object-Oriented Programming in Java

Java is fundamentally object-oriented. Key OOP concepts include:

  • Classes and objects
  • Inheritance
  • Polymorphism
  • Abstraction
  • Encapsulation

// OOP Example in Java
class Animal {
    String name;
    
    public Animal(String name) {
        this.name = name;
    }
    
    public void sound() {
        System.out.println("Makes a sound");
    }
}

class Dog extends Animal {
    public Dog(String name) {
        super(name);
    }
    
    @Override
    public void sound() {
        System.out.println("Bark");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal myDog = new Dog("Buddy");
        myDog.sound();  // Outputs: Bark
    }
}
