Easy

function factorial(n){

    return n * factorial(n-1);
}

C++: Palindrome Check

Medium

Check if a given string is a palindrome in C++ without using extra space.

#include <string>
bool isPalindrome(const std::string &s){
    int l=0, r=s.size()-1;
    while(l<r){
        if(s[l++] != s[r--]) return false;
    }
    return true;
}

Rust: Fibonacci

Medium

Write a function in Rust to return nth Fibonacci number using recursion.

fn fib(n:u32) -> u32 {
    if n==0 || n==1 { return n; }
    fib(n-1) + fib(n-2)
}
